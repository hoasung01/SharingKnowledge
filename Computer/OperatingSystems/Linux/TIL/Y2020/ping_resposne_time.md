```
1. < 30ms
  - excellent ping; almost unnoticeable; ideal for online gaming
2. 30-50ms
  - average ping; still ok for online gaming
3. 50-100ms
  - somewhat slow ping time; not too noticeable for web browsing
  but may affect gaming
4. 100-500ms
  -  slow ping; minimal effect on web browsing, but will create
  noticeable lag in online gaming
5. > 500ms
  - pings of a half second or more will add a noticeable delay
  to all requests; typically happens when the source and destination
  are in different parts of the world
```
